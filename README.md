# 📰 Fake News Detector

A machine learning web application that detects whether a news 
article or headline is REAL or FAKE using Natural Language 
Processing (NLP) and Logistic Regression.

## 🌐 Live Demo
[Click here to try the app!](https://neha2006-6-fake-news-detector.streamlit.app)

## 📌 About the Project
In today's world, fake news spreads faster than real news.
This project aims to combat misinformation by automatically 
classifying news articles as real or fake using Machine Learning.

## ✨ Features
- Paste any news headline or article
- Instantly detects Real or Fake
- Shows confidence percentage
- Simple and clean UI

## 🛠️ Technologies Used
- Python
- Scikit-learn (TF-IDF + Logistic Regression)
- NLTK (Natural Language Processing)
- Streamlit (Web App)
- Pandas (Data Processing)

## 📊 Model Performance
- Algorithm: Logistic Regression
- Vectorization: TF-IDF (5000 features)
- Accuracy: ~95%
- Dataset: 40,000+ news articles

## 🚀 How to Run Locally
pip install -r requirements.txt
streamlit run app.py
